### 🛡️ Kalkan — Document Integrity Layer for AI Agents
<p align="center">
  An MCP layer that stops AI agents from silently corrupting your documents — the model can only <em>propose</em> validated, versioned patches, and every rollback is a byte-exact file restore. An open-source response to Microsoft Research's <b>DELEGATE-52</b> finding that LLMs silently corrupt ~25% of a document over long delegated editing sessions.
</p>
<p align="center">
  <b>Direct AI editing corrupts 30% of a document — Kalkan measures 0%.</b>
</p>
